# scaletoscrape

# Overview

In this assignment, we are creating an application that allows users to view and leave comments on articles that we scraped from an URL. 

# System Build Foundation

* Express
* Express-Handlebars
* Mongo
* Mongoose
* Cheerio
* Axios
* Morgan
* Heroku
